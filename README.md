# 🎮 Real Money Game App

This is a full-featured **Real Money Game App** built using **React Native + Firebase**.  
It supports user registration, login, deposits via Bkash, withdrawals, game access, referral, and real-time transaction history.

---

## 📱 Features

- 🔐 Authentication (Login, Register, Forgot Password)
- 💰 Real-time Balance View
- 🎮 Game Selection (LUDO, CARROM, CHESS, BEAD 16)
- 📤 Bkash Deposit System
- 📥 Withdraw System
- 👤 Profile Management + Profile Picture Upload
- 📜 Transaction History
- 📞 Contact Support (Facebook, WhatsApp, Telegram)

---

## 🚀 Technology Stack

- 📱 **React Native (Expo)**
- 🔥 **Firebase (Authentication, Firestore, Storage)**
- 🧭 **React Navigation**

---

## 📦 Installation (For Developers)

### Prerequisites
- Node.js + npm
- Expo CLI
- Firebase Project
- `google-services.json` in project root

### Install Dependencies

```bash
npm install
npx expo install @react-navigation/native
npx expo install @react-navigation/native-stack
npx expo install react-native-screens react-native-safe-area-context react-native-gesture-handler react-native-reanimated
